### Installation

Dillinger requires [Node.js](https://nodejs.org/) v10+ to run.

Instalando as dependências e dependências de desenvolvimento para executar o sistema

Executando aplicação back-end

```sh
$ cd ignite-challenge-01
$ yarn
$ yarn dev
```
